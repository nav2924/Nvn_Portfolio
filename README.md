# 🚀 3D Developer Portfolio — Built with Three.js, GSAP & React

This is a stunning, responsive, and animated developer portfolio built using **React**, **@react-three/fiber**, **Three.js**, **GSAP**, and **Tailwind CSS**. It features smooth 3D model interactions, animated counters, and an elegant layout to showcase your developer journey in a modern way.

![Hero Section](![image](https://github.com/user-attachments/assets/03e5b15b-7ade-48c2-b902-68a6a7cd7fe3))

---

## 🔮 Features

- 🎨 Interactive 3D Model Experience (using `@react-three/fiber` + Drei)
- ✨ Smooth animations with `GSAP` and `ScrollTrigger`
- 📱 Fully Responsive across all devices
- 🧮 Animated Counter Section
- 🌗 Dark Mode Ready (optional)
- 🧠 Clean and modular code structure

---

## 📦 Tech Stack

| Tech            | Description                                 |
|-----------------|---------------------------------------------|
| React           | JavaScript Library for UI                   |
| Tailwind CSS    | Utility-First CSS Framework                 |
| Three.js        | 3D Graphics Library (via `@react-three/fiber`) |
| GSAP + ScrollTrigger | Animation Platform for smooth transitions |
| @react-three/drei | Helper components for easier 3D setup     |
| React Responsive | Media Query Handling for 3D Scaling        |
